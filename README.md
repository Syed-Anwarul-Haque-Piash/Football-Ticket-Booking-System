# Football Ticket Booking System - Database Design & SQL Queries

## Project Overview

This project implements a **Football Ticket Booking System Database** designed to manage football fans purchasing tickets, upcoming tournament matches, and individual ticket booking receipts. The system demonstrates intermediate-to-advanced SQL concepts and proper database design principles.

 **Database Design**: Entity Relationship Diagram (ERD) with 1-to-1, 1-to-Many, and Many-to-1 relationships
 **Relational Integrity**: Primary keys, foreign keys, and referential integrity constraints
 **SQL Fundamentals**: CREATE TABLE, INSERT, SELECT, WHERE clauses
 **Data Validation**: CHECK constraints, NOT NULL, UNIQUE, DEFAULT values
 **Advanced SQL Queries**:
  - JOIN variants (INNER JOIN, LEFT JOIN)
  - Subqueries and aggregation functions
  - Pattern matching (LIKE, ILIKE)
  - NULL handling (IS NULL, COALESCE)
  - Pagination (LIMIT, OFFSET)
  - ORDER BY and sorting
